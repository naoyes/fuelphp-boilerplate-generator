# fuelphp-boilerplate-generator

## Generate Boierplate
```
$ bundle --path vendor/bundle
$ mkdir -p /some/where
$ bundle exec rake appname="myapp" path="/some/where/app"
```

## Deploy App
```
$ bundle --path vendor/bundle
$ mkdir -p /some/where
$ bundle exec rake deploy repo="git://example.com/hoge.git" path="/some/where/app"
```
